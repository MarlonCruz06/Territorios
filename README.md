Programa de cadastro de territórios

Como compilar (Windows, se tiver GCC instalada - MinGW/MSYS2):

gcc -Wall -Wextra -o territorios.exe territorios.c

Como executar:

.\territorios.exe

Notas:
- O programa cadastra 5 territórios (nome, cor do exército, tropas) e exibe
  os dados logo em seguida.
- Use entradas simples; o nome e a cor aceitam espaços (limitados pelos
  tamanhos definidos).
